# PrestaShop Currencies API

Small php app that fetches currency rate from [Fixer.io](https://fixer.io/) and outputs a XML file

## Usage

No need to install dependencies

```
php fetchLatestRates.php <API_KEY> <OUTPUT_FILE>
```
